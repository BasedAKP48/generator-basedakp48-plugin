# @basedakp48/generator-plugin

> Scaffold out a [BasedAKP48](https://github.com/BasedAKP48) plugin.

## Install

```
$ npm install --global @basedakp48/generator-plugin
```


## Usage

With [yo](https://github.com/yeoman/yo):

```
$ yo @basedakp48/plugin
```
